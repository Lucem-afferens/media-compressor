Тонкие настройки (ffmpeg)

CRF: меньше число — лучше качество, больше файл (обычно 18–28). H.265 даёт меньший размер, но кодируется дольше. Preset «slow» — ещё меньше файл при том же CRF.

    Сжатие и конвертация аудио

    Уменьшите размер трека, смените формат или вытащите звук из видео — MP4, MOV, MKV и другие ролики поддерживаются.

    Вход
    MP3, WAV, FLAC, M4A, OGG, Opus · или видео (извлекается первая аудиодорожка)
    Выход
    MP3 — меньше размер · M4A — Apple · Opus — голос · FLAC — без потерь
    1. Загрузите аудио или видеофайл
    2. Выберите формат — пресет «Из видео → MP3» для роликов
    3. Нажмите «Сжать и скачать» (для видео — «Извлечь звук и скачать»)

    Аудио или видео — сожмём или извлечём звук

    MP3, WAV, FLAC… · MP4/MOV/MKV — звук извлекается автоматически · до {{ max_upload_human }}

    Для роликов MP4/MOV: извлекается звук и сжимается в MP3 192 kbps.

    Тонкие настройки (битрейт, частота)

    MP3 192 kbps — универсально. Opus — лучше для голоса при меньшем размере. FLAC — без потерь, файл крупнее. VBR для MP3 подбирает битрейт по сложности звука.

    Текст из аудио и видео

    Распознавание речи и песен с таймкодами — предложения для речи, строки для песен. Экспорт TXT, SRT, VTT, LRC, JSON.

    Режимы
    Речь — предложения · Песня — строки · Авто — по эвристике
    Экспорт
    ZIP: transcript.txt, subtitles.srt, subtitles.vtt, lyrics.lrc (песни), transcript.json
    1. Загрузите аудио или видео
    2. Выберите режим и язык; для песен — исполнитель/название (Genius)
    3. Дождитесь результата — клик по строке перематывает плеер
